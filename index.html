Revealer

Flask app to serve and control Reveal.js slideshows

View on GitHub Download .zip Download .tar.gz

Present to your audience

Revealer allows to serve multiple Reveal.js presentations using an http server to navigate and select currently active slideshows. Users can comment and review on active or inactive presentations, active presentations reviews can be read by the presenter.

Add resources to your slideshows

Presentations can use any web resource, either by putting links in their code or by linking the files uploaded along with the presentation.

Editing presentations

Right now, editing already uploaded presentations is not supported, nevertheless, including text and revealer editor is currently in progress.
